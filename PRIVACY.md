# Privacy Policy

**Fast Java Thread** is a fully offline desktop application. It does not:

- Collect any user data
- Send any data over the network
- Use analytics, telemetry, or tracking
- Load any external resources (CDN, fonts, scripts)
- Require an internet connection to operate

All thread dump analysis is performed locally on your machine. Your data never leaves your computer.

## MCP Server

The optional MCP server runs entirely on your local machine. When using HTTP transport, it listens only on `localhost` and is not accessible from other devices. No thread dump data is transmitted externally — MCP clients connect to the local server and all processing stays on your computer.
